# Travel Booking Website

A modern flight and hotel booking website built with React.

## Features

- Flight booking with One-Way, Round Trip, and Multi-City options
- Top flight deals section
- Popular airlines showcase
- Hotel booking section
- Responsive design
- Tab-based navigation (Flights, Hotels, Trains, Buses, Cabs)

## Installation

```bash
cd travel-booking/frontend
npm install
npm run dev
```

The app will run at http://localhost:3000

## Project Structure

- Header with navigation tabs
- Hero section with booking form
- Flight deals section
- Popular airlines section
- Hotel booking section
