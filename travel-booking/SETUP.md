# Travel Booking Website Setup

## Quick Start

```bash
cd travel-booking/frontend
npm install
npm run dev
```

Open http://localhost:3000

## Features Implemented

✅ Header with tabs (Flights, Hotels, Trains, Buses, Cabs)
✅ Hero section with flight booking form
✅ Trip type tabs (One-Way, Round Trip, Multi-City)
✅ Top Flight Deals section with cards
✅ Most Popular Airlines section
✅ Book Your Hotel section
✅ Responsive design
✅ Clean, modern UI matching the design

## Components

- **Header**: Navigation with tabs and Sign In button
- **HeroSection**: Main booking form with airplane image
- **FlightDeals**: Top deals cards
- **PopularAirlines**: Airline cards grid
- **HotelBooking**: Hotel cards with ratings

## Customization

- Colors can be adjusted in CSS files
- Add real images by replacing placeholders
- Connect to backend API for real data
- Add more functionality to booking forms
